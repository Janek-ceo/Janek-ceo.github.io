Kurs: Jak programować w HTML i CSS

HTML (HyperText Markup Language) i CSS (Cascading Style Sheets) to podstawowe technologie używane do tworzenia stron internetowych. Oto kilka kroków, które pomogą Ci rozpocząć naukę:

1. Podstawy HTML

HTML jest językiem znaczników, który służy do strukturyzacji treści na stronie. Oto przykładowa struktura dokumentu HTML: 

<!DOCTYPE html>
<html lang="pl">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Tytuł Strony</title>
</head>
<body>
    <h1>Witaj w HTML!</h1>
    <p>To jest przykładowy paragraf.</p>
</body>
</html>

2. Podstawy CSS

CSS jest używany do stylizacji stron. Możesz zmieniać kolory, czcionki, układ i wiele innych. Oto przykład, jak za pomocą CSS zmienić kolor tekstu: 

p {
    color: blue;
}

3. Łączenie HTML i CSS

Możesz połączyć HTML i CSS na dwa sposoby: za pomocą stylów wbudowanych w tagu HTML lub za pomocą zewnętrznych arkuszy stylów. Oto przykład zewnętrznego arkusza stylów: 

<link rel="stylesheet" href="style.css">

To wszystko, aby rozpocząć swoją przygodę z programowaniem w HTML i CSS! Pamiętaj, aby regularnie ćwiczyć i eksperymentować z kodem.
